***********************
*     Description     *
***********************
 * Blowfish Password Manager can be used for encryption or decryption 
 * via blowfish algorithm on SSC Applications (like db password on installers)
 *
 * Compiled by : Alper Tunaboylu, Fatih Kara 
 * Compiled at : 14/03/2018 in Windows 7 OS with Java 1.7

*****************
*     Usage     *
*****************
 * Sample Encrypt Usage: java -jar BlowfishPasswordManager e samplePlainText
 * Sample Decrypt Usage: java -jar BlowfishPasswordManager d sampleEncryptedText
